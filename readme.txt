Send notifications of new questions and answers to a Slack channel.

== Description ==
Now with DWQA Slack Integration, your Slack group will be notified whenever there are new questions are created on your question page. 

DWQA Slack Intergration is an extension of DW Question & Answer. It uses the slack bot to keep track of your forum page and continuously send notification to your Slack Group Chat. There is one requisite that is the forum must use DW Question & Answer.

= Key Features =
* Notify via Slack when have new question, answer or comment on your site
* Option on/off push notify

== Installation ==

1. Unpack the entire contents of this plugin zip file into your `wp-content/plugins/` folder locally
1. Upload to your site
1. Navigate to `wp-admin/plugins.php` on your site (your WP Admin plugin page)
1. Activate this plugin

OR you can just install it with WordPress by going to Plugins > Add New > and type this plugin\'s name

== Frequently Asked Questions ==
= Setup A New Slack WebHook =
1. Go To `https://yourteam.slack.com/services/new/incoming-webhook`
1. Create a new webhook
1. Set a channel to receive the notifications
1. Copy the URL for the webhook

= Setting Up The Plugin =
1. Go to Dashboard > Questions > Slack Integration
1. Paste the webhook URL into the form.
